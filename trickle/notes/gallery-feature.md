# Галерея изображений товаров

Дата создания: 2025-11-27

## Описание функции

Реализована возможность добавления нескольких фотографий для каждого товара с отображением галереи в карточке товара.

## Возможности

### В карточке товара:
- Переключение между фотографиями стрелками влево/вправо
- Точки-индикаторы текущего изображения
- Счетчик фотографий (например, "2 / 5")
- Плавная анимация перехода между фото

### В админ-панели:
- Поле для добавления основного изображения (обязательное)
- 4 отдельных поля для дополнительных изображений (опциональные)
- Бейдж с количеством фото на миниатюре товара в списке

## Как использовать

1. В админ-панели откройте форму редактирования/создания товара
2. Укажите URL основного изображения
3. В полях "Дополнительные изображения" добавьте URLs других фото (максимум 4)
4. Сохраните товар

## Структура данных

Товар теперь имеет два поля для изображений:
- `image` - основное изображение (string, обязательное)
- `images` - массив дополнительных изображений (array of strings, максимум 4)

## Технические детали

- Изображения оптимизируются автоматически через `imageOptimizer`
- Галерея работает на touch-устройствах (можно листать свайпом)
- Первым всегда показывается основное изображение (`image`)
- При закрытии модального окна индекс изображения сбрасывается на первое
